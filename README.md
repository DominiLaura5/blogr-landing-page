# Blogr Landing Page

A responsive landing page for **Blogr** — a modern publishing platform.
Built as a Frontend Mentor challenge.

## Live Demo

[View Live Site](https://blogr-landing-page-1.vercel.app/)

## Built With

- HTML5 & CSS
- JavaScript (ES6+)

## Features

- Fully responsive (mobile, tablet, desktop)
- Animated navigation with dropdown menus
- Mobile hamburger menu
- Smooth scroll animations
- Clean, accessible HTML structure
